Tranches de Vie

Posted on 2013, Feb 06 One min read

I enhanced my video processing skills with the artistic installation “Tranche de vie”” (built with OpenCV). I developed this project with openframeworks for a large LED screen.

One of the goal of the installation was to use the special format of the screen which was composed of 5 large vertical stripes.
